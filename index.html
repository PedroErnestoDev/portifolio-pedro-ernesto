IP Address Tracker

Este projeto é um rastreador de endereços IP. Ele permite que você insira um IP ou endereço de domínio e veja sua localização em um mapa interativo. Para obter as localizações dos IPs, foi utilizada a API de Geolocalização IP da IPify. Para gerar o mapa, foi usada a biblioteca LeafletJS. O objetivo deste projeto é replicar o design de referência o mais próximo possível, e você pode experimentar qualquer ferramenta ou recurso adicional que quiser para aprimorá-lo.
